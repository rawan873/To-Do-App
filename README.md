# To-Do-App
A JavaScript To-Do App with task management, progress tracking, Local Storage persistence, and completion celebrations.
A modern To-Do List web application built with HTML, CSS, and JavaScript.

Features:
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Progress tracking with a dynamic progress bar
- Task completion statistics
- Local Storage support (tasks persist after page refresh)
- Celebration confetti when all tasks are completed

This project was created to practice DOM manipulation, event handling, array methods, and Local Storage in JavaScript.
